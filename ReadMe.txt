เกม Whac A Mole
วิธีเล่น
1. เข้าเกม <หน้า Menu> ใช้ Keyboard ลูกศรขึ้น – ลง เพื่อเลือก ดังนี้ 
 a.<Start>  เริ่มเล่นเกม
 b.<Exit> ปิดเกม
2. เมื่อ <Start>  เริ่มเล่นเกม จะเข้าเล่นใน 2 States
 a.State 1 ทุบตุ่น 6 ตัว โดยกด ปุ่ม S D F - > ตุ่นด้านล่าง J K L -> ตุ่นด้านบน ได้แต้ม 10 คะแนน/ครั้ง
 b.State 2 ทุบตุ่นโบนัส ได้แต้ม 50 คะแนน/ครั้ง
หมายเหตุ ในการเล่นจะจับเวลาแต่ละ  State ไว้เพื่อ ตั้งเวลาในการเล่นแต่ละ State

อธิบาย รายละเอียด Class
1. Game Class -> เป็น Main class ของเกม
	- สร้าง JFrame ขึ้นมาเพื่อใส่ GamePanel ให้มารัน

2. GamePanel Class
	- สร้าง JPanel ขึ้นโดย เพิ่มการทำงาน ActionListener
	- รับอินพุตจากคีย์บอร์ด ด้วย KeyListener
	- สร้าง Timer โดยการใช้ตัว class นี้ เพื่อกำหนดการเรนเดอร์การแสดงผล
	- สร้าง GameStateManager ขึ้นมาเพื่อการควบคุมการเล่นเกมในแต่ละ State

3. GameStateManager Class
	- สร้างเพื่อการควบคุมการเล่นเกมว่า ทำงานไปแต่ละ State อย่างไร

4. GameState Class
	- เป็นคลาสแม่เพื่อสร้าง State ต่างๆให้ GameStateManager ควบคุม

5. MenuState Class
	- สร้างหน้าเมนู และเลื่อกการทำงานโดยรับอินพุตจากคีย์บอร์ด

6. Level1State Class
	- เกม State 1 เป็นการทุบตุ่นตัวเล็กทั้งหมด 6 ตัว โดยการกดปุ่มคีย์บอร์ดที่กำหนด และจะได้คะแนน

7. Level2State Class
	- เกม State 2 เป็นการทุบตุ่นยักษ์โดยการกดปุ่มคีย์บอร์ดที่กำหนด และจะได้คะแนน

8. Sprite Class
	-คลาสแม่ของตัวตุ่นทั้งหมด

9. BigMole Class
	- คลาสของตัวตุ่นยักษ์ กำหนดการทำงานต่างๆ
	- เช่น ทำงานตามปุ่มที่กำหนดอย่างไร ให้คะแนนเท่าไหร่

10. SmallMole Class
	- คลาสของตัวตุ่นปกติ กำหนดการทำงานต่างๆ
	- เช่น ทำงานตามปุ่มที่กำหนดอย่างไร ให้คะแนนเท่าไหร่ สุ่มโอกาสที่จะโผล่ขึ้นมา สุ่มความเร็วในการเคลื่อนที่
	- เป็นคลาสแม่ของ LowerMole และ UpperMole

11.	LowerMole Class
	- คลาสของตัวตุ่นปกติที่โผล่จากด้านล่าง กำหนดการทำงานต่างๆ
	- เช่น มีการเคลื่อนไหวขึ้น-ลงอย่างไร 

12.	UpperMole Class
	- คลาสของตัวตุ่นปกติที่โผล่จากด้านบน กำหนดการทำงานต่างๆ
	- เช่น มีการเคลื่อนไหวขึ้น-ลงอย่างไร
